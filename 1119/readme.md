# AI EXPO KOREA 2026 🤖

> AI 기업의 가장 FIT한 B2B 마케팅 솔루션

PWA(Progressive Web App)와 QR 코드를 활용한 AI EXPO KOREA 2026 공식 랜딩 페이지입니다.

## 🌐 라이브 데모

**[https://gwonbubjin.github.io/Hybrid/AI-EXPO-2026/](https://gwonbubjin.github.io/Hybrid/AI-EXPO-2026/)**

<br>

## ✨ 주요 기능

- 📱 **PWA 지원** - 오프라인 접근 가능, 홈 화면 설치
- 📲 **QR 코드** - PC에서 모바일로 빠른 접속
- 💻 **반응형 디자인** - 모바일/PC 최적화
- ⏱️ **실시간 카운트다운** - 행사까지 남은 시간 표시
- 🖼️ **배경 슬라이드쇼** - 자동 전환 이미지
- 🌍 **다국어 지원** - 한국어/English

<br>

## 🛠️ 기술 스택

- HTML5, CSS3, JavaScript
- PWA (Service Worker, Web App Manifest)
- QRCode.js
- GitHub Pages

<br>

## 📂 프로젝트 구조

```
AI-EXPO-2026/
├── index.html          # 메인 페이지
├── style.css           # 스타일시트
├── sw.js               # Service Worker
├── manifest.json       # PWA 매니페스트
├── poster.jpg          # 포스터 이미지
├── 1.jpg               # QR 코드
├── bg1.jpg, bg2.jpg, bg3.jpg  # 배경 이미지
└── icon-192.png, icon-512.png # PWA 아이콘
```

<br>

## 🚀 로컬 실행

```bash
# 저장소 클론
git clone https://github.com/gwonbubjin/Hybrid.git
cd Hybrid/AI-EXPO-2026

# 로컬 서버 실행
python -m http.server 8000

# 브라우저에서 열기
# http://localhost:8000
```

<br>

## 📱 PWA 설치

- **Android**: 메뉴(⋮) → "홈 화면에 추가"
- **iOS**: 공유(⬆️) → "홈 화면에 추가"
- **PC**: 주소창 설치 아이콘(⊕) 클릭
