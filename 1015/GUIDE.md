## 📝 프롬프트 사용 방법

### STEP 1: AI 사이트 접속
- [Claude.ai](https://claude.ai) 또는
- [ChatGPT](https://chat.openai.com) 접속

### STEP 2: 프롬프트 복사
- 아래 프롬프트를 **전체 복사** (Ctrl+C 또는 Cmd+C)

### STEP 3: AI에게 질문
- 채팅창에 **붙여넣기** (Ctrl+V 또는 Cmd+V)
- **Enter** 누르기

### STEP 4: 코드 저장
- AI가 만들어준 코드를 **복사**
- 메모장이나 VS Code에 **붙여넣기**
- **저장** (파일명은 프롬프트에 적혀있음)

### STEP 5: 확인
- 웹 브라우저에서 `index.html` 파일 열기
- 결과 확인!

### 프롬프트 1: 가장 기본이 되는 HTML 파일 만들기
```

   - 제목: "국립중앙박물관 - 한국의 역사와 문화"

4. <body> 안에:
   - 헤더(header)
   - 메인(main)
   - 푸터(footer)

   3f: { title: '3F', facilities: ['전시영역', '어린이박물관', '특별전시설1'] }
   (나머지 층도 추가)
5. 탭 클릭하면:
   - 모든 탭에서 .active 제거
   - 클릭한 탭에만 .active 추가
   - data-floor 값 가져오기 (예: "3f")
   - 이미지 src 변경: images/layout/floors/3f.png
   - 정보 텍스트 변경

주석 달아서 코드를 완성해줘.
```

### 프롬프트 30: 이미지 확대 모달 만들기
```
이미지를 클릭하면 크게 보여주는 모달 기능을 만들어줘.

함수 이름: initImageZoom()

해야 할 일:
1. 모달 찾기 (id="imageModal")
2. 모달 안의 이미지 찾기 (id="modalImage")
3. 설명 찾기 (id="modalCaption")
4. 닫기 버튼 찾기 (.modal-close)
5. 확대 가능한 모든 이미지 찾기 (.image-zoomable)
6. 이미지 클릭하면:
   - 모달에 .active 추가 (보이게)
   - 클릭한 이미지의 src를 모달 이미지에 복사
   - alt를 설명에 복사
   - body에 overflow: hidden (스크롤 막기)
7. 모달 닫기 (3가지 방법):
   - 닫기 버튼 클릭
   - 검은 배경 클릭
   - ESC 키 누름
8. 닫을 때:
   - .active 제거
   - overflow: auto (스크롤 복원)

주석 많이 달아서 코드를 완성해줘.
```

### 프롬프트 32: 스크롤 진행 바 만들기
```
페이지를 스크롤하면 상단에 진행 바가 채워지는 기능을 만들어줘.

해야 할 일:
1. HTML에 진행 바 추가:
   <div id="progress-bar"></div>
2. CSS로 스타일:
   - position: fixed, top: 0
   - 높이: 4px
   - 배경: 금색
   - 처음엔 너비: 0%
3. JavaScript로 스크롤 감지:
   - 전체 높이 계산
   - 현재 스크롤 위치 계산
   - 퍼센트로 진행 바 너비 변경

주석 달아서 전체 코드(HTML + CSS + JS)를 만들어줘.
```

### 프롬프트 33: 다크 모드 토글 만들기
```
버튼을 클릭하면 다크 모드로 바뀌는 기능을 만들어줘.

해야 할 일:
1. HTML에 토글 버튼 추가
2. CSS 변수를 다크 모드용으로 추가:
   [data-theme="dark"] {
     --primary-dark: #ffffff;
     --primary-light: #1a1a1a;
     (색상 반전)
   }
3. JavaScript:
   - 버튼 클릭 시 <html>에 data-theme="dark" 추가/제거
   - localStorage에 저장 (새로고침해도 유지)

주석 달아서 전체 코드를 만들어줘.
```
